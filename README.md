# cpp_exercism

My solutions for the **Exercism.org C++ track**.

## Repository layout

Solutions live under:

- `solutions/cpp/<exercise-slug>/`

Examples (from this repo):

- `solutions/cpp/hello-world/`
- `solutions/cpp/atbash-cipher/`
- `solutions/cpp/raindrops/`
- `solutions/cpp/leap/`

## Notes

- These are **learning-focused** solutions; they may evolve as I revisit problems.
- Exercise descriptions and test suites are provided by Exercism; this repository contains **my code only**.

## Running / testing locally (typical Exercism workflow)

From an exercise directory:

1. Open the exercise folder (e.g. `solutions/cpp/hello-world/`)
2. Build & run using your preferred C++ toolchain / IDE
3. Run the provided tests (Exercism supplies tests for each exercise)
